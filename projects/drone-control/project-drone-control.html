Motion Tracking Drone Control

This project developed a real-time drone control system using webcam-based hand motion tracking, translating hand gestures into drone commands via the Crazyflie Python API. Key achievements included integrating hand tracking with drone control, optimizing data communication, and implementing advanced maneuvers with enhanced stability and responsiveness.

Development Process

Establishing Basic Control

The initial phase of the project focused on establishing a fundamental connection between hand tracking and drone control. This involved:

  • Establishing communication between the hand tracking system and the drone's API.
  • Implementing simple thrust control, allowing the drone to move only up and down based on hand gestures.
  • Operating without a sophisticated stabilization system, which provided foundational insights into real-time control challenges.

This early test, while rudimentary, proved the feasibility of using hand gestures for direct drone manipulation and laid the groundwork for future development in stability and complex maneuver implementation.
